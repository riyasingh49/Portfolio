# Riya Singh | Developer Portfolio

This is my personal developer portfolio website. It presents my introduction, achievements, certifications, tech stack, projects, experience, and contact links in a responsive single-page layout.

🔗 [View Live Portfolio](https://portfolio-nine-gamma-dcv64029u5.vercel.app)

---

## Project Tech Stack

This portfolio project is built with:

- React.js
- JavaScript
- HTML
- CSS


---

## Features

- Responsive personal portfolio layout
- About section with skills and achievements
- Certifications showcase
- Tech stack section
- Project showcase with hover descriptions and project links
- Experience section
- Contact section with email, GitHub, and LinkedIn links
- Smooth section navigation

---

## Run Locally

```bash
git clone https://github.com/riyasingh49/Portfolio.git
cd Portfolio
npm install
npm run dev
```

## Build

```bash
npm run build
```
